## 与当前项目的关系

当前 Kiro 快捷键盘项目已选用：
- **按键屏**：微雪 0.85寸 ScreenKey Module B (ST7735, 128×128) × 3 — 淘宝价 ¥29-30/个
- **圆形屏**：微雪 0.71寸 LCD Module (GC9D01, 160×160) × 1 — 淘宝价 ¥82
- **矩形屏**：微雪 1.47寸 LCD Module (ST7789V3, 172×320) × 1 — 淘宝价 ¥57

如需扩展或替代方案：
- 圆形屏可考虑 GC9A01 1.28寸（¥14-16，分辨率 240×240，更大更清晰）
- 长条形屏 NV3007 2.79寸（¥7-10）可用于未来的进度条/状态栏显示
- 更大的 1.8寸圆形触摸屏 ST77916（¥22.8, 360×360）适合做旋钮交互
