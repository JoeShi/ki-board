## 七、竞争格局与机会

### 头部玩家

| 玩家 | 定位 | 优势 | 劣势 |
|------|------|------|------|
| ami.moment | KOL + 低成本方案 | 27k赞，极高影响力，¥250低门槛 | 非自研产品，组合现成设备 |
| 小皮鯊魚 | 极客/开发者 | 技术深度高，Agent切换+语音，计划开源 | TP-7太贵(¥10000)，非大众方案 |
| Ulanzi (优篮子) | 商业品牌 | 唯一正式商业化的，品牌力+供应链 | 产品未上市，进度不明 |
| GeekLogic | 开源极客 | 完整开源（代码+建模），社区友好 | 更偏概念验证，产品化不足 |
| Elgato Stream Deck | 成熟品类 | 生态完善，有 Claude Code 插件 | 非专为 AI coding 设计，价格偏高 |

### 市场机会

1. **空白点**：没有一款 ¥200-500 价位、即插即用的 Vibe Coding 专用硬件
2. **核心痛点**：
   - 语音输入触发/停止需要物理按键（系统快捷键不方便）
   - 多 Agent 切换频繁
   - 需要状态可视化（AI 在跑什么、花了多少额度）
3. **差异化方向**：
   - 带屏幕的按键（vs 翰林阅无屏翻页器）
   - Agent 状态实时显示（vs Stream Deck 静态图标）
   - USB 即插即用（vs 蓝牙配对/WiFi配置）
   - 开源 + 可自定义（vs Ulanzi 闭源商业产品）

### 用户画像

- **核心用户**：使用 Claude Code / Cursor / Kiro 的开发者和非技术 Vibe Coder
- **痛点共鸣**：评论区高频关键词 = "在哪买"、"开源吗"、"什么时候卖"、"能不能不用打字"
- **预算敏感度**：¥20-250 的 DIY 方案点赞远超万元方案，说明大众用户对价格敏感
- **平台偏好**：macOS 用户为主（多条笔记提到 Mac 原生、Ghostty）

## 八、对 Kiro 键盘项目的启示

### 产品定位优势

我们的 Kiro 键盘（ESP32-S3 + 5屏 + USB HID）恰好填补了市场空白：

| 维度 | 市场现状 | Kiro 键盘 |
|------|----------|-----------|
| 带屏按键 | 仅 Stream Deck（贵）或无屏翻页器 | ✅ 3个128x128屏幕按键 |
| Agent 状态显示 | 无现成方案 | ✅ 圆屏表情 + 矩形屏4-Agent总览 |
| 即插即用 | 多数需要配置 | ✅ USB HID，无需驱动 |
| 语音输入 | 需额外麦克风 | ✅ 系统语音触发 + companion 3P ASR |
| 价格定位 | 缺少 ¥200-500 方案 | 目标 BOM ¥150-300 |
| 开源 | 少数项目开源 | ✅ 完全开源 |

### 内容营销建议

1. **首发内容方向**：DIY制作过程 + 实际使用场景视频（参考 ami.moment 的 27k 赞模式）
2. **关键标签**：#vibecoding #键盘 #先进生产力 #claude #DIY #开源
3. **差异化卖点**："不只是按键——它能看到你的 AI 在做什么"
4. **价格锚点**：对标 TP-7(¥10000) 和 Stream Deck(¥1500)，强调 "1/10 价格，更专业"
