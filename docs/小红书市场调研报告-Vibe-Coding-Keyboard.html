五、是否有人在卖？

类型详情
正在商业化的Ulanzi MC-01 "AI指挥台"（优篮子官方，麦克风+小键盘一体，正在开发中，评论区大量催更"什么时候开卖"）
带店铺引导的翰林阅翻页器（ami.moment 笔记自带产品名，评论区问购买链接）；DJI Mic Mini（同上）
开源可复刻的GeekLogic 键盘（嘉立创PCB+3D打印，全部开源）；工科男孙老师键盘（海思平台开源）

结论：目前市场上没有成熟的 "Vibe Coding 专用键盘" 商品在售。Ulanzi 是唯一正式宣布在做商业产品的公司，但尚未上市。这是一个明确的市场空白。

市场机会

空白点
没有一款 ¥200-500 价位、即插即用的 Vibe Coding 专用硬件
用户预算
¥20-250 的 DIY 方案点赞远超万元方案，大众用户价格敏感

核心痛点（评论区高频提及）

  • 语音输入触发/停止需要物理按键（系统快捷键不方便）
  • 多 Agent 切换频繁，缺少物理控制
  • 需要状态可视化（AI 在跑什么、花了多少额度）
  • "在哪买"、"开源吗"、"什么时候卖" — 需求明确但无产品可买

用户画像

  • 核心用户：使用 Claude Code / Cursor / Kiro 的开发者和非技术 Vibe Coder
  • 平台偏好：macOS 用户为主（多条笔记提到 Mac 原生、Ghostty）
  • 消费意愿：愿意为"好用的工具"付费，但对>¥500的专用外设犹豫
